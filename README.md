# Interactive-UI
A UI showcase to show sample interactive elements

## Specifications
 - Webpack 4.x
 - React 16.x.x
 - Material-UI

# Requirements
 - Node.js - latest v8.x

## Installation
 - `npm install` - Node modules

## Usage
 - `npm run dev` - Run with Webpack dev server
 - `npm run build` - Build production-ready code
 - `npm run lint` - Run linter
